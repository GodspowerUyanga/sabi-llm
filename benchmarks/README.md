# Benchmarks

This folder holds the local benchmark prompt set and generated reports.

- `prompts.jsonl` — one JSON object per line. Fields: `name`, `intent`
  (`THINK` / `CODE` / `CHAT`), `prompt`, and `expected_keywords` (used as a
  lightweight accuracy proxy).
- `report.json` / `report.md` — generated by `python scripts/run_benchmark.py`.

## Run

```bash
sabi benchmark                 # quick summary in the terminal
python scripts/run_benchmark.py   # writes report.json + report.md
```

## Scoring dimensions (ADTC 2026)

| Dimension | Weight | How SABI measures it |
|-----------|:------:|----------------------|
| Accuracy  | 50%    | keyword-coverage proxy over the prompt set |
| Speed     | 30%    | tokens/sec during generation |
| Efficiency| 20%    | `100 × (7 GB − peak RSS) / 7 GB` |

A −10 thermal penalty applies if core temperature exceeds 85 °C.
